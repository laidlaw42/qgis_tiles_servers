# Basemap XMLs for QGIS
List of XYZ tiles for importing into QGIS without the use of plugins.
The ArcGIS Rest Servers are very much NSW ecology-centric.


## Usage
In the Browser panel:
- Secondary click the server type (XYZ Tiles, Vector Tiles, ArcGIS REST Serves)
- Load Connections...
- Navigate to the respective XML from this repo
- Import whichever base layers you want
